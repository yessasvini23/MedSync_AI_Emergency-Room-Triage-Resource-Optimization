# MedSync AI - Emergency Response System

A real-time triage and hospital routing system powered by AI.

## Features
- Real-time emergency case triage
- AI-powered patient severity assessment
- Hospital network optimization
- 4-hour capacity forecasting
- Emergency response command center

## Local Setup

```bash
pip install -r requirements.txt
streamlit run medsync_ai.py
```

## Deployment

Deployed on [Render.com](https://render.com)

## Technologies
- Streamlit
- Python
- Pandas
- Plotly
